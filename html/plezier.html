Makey Makey Opdracht

De makey makey opdracht was een opdracht voor software development practice lessen. Het is een project waar ik heel erg van heb genoten we mochten dit project namelijk samen doen. We hebben een makey makey gebruikt om een kleine controller te maken die vergelijkbaar aan een wii controller werkt deze controller heeft namelijk een zijwaartse input die bijvoorbeeld te gebruiken is om een poppetje van links naar rechts te bewegen door de controller naar links of rechts' onder te bewegen.
